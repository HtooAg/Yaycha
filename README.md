# Yaycha App

**Yaycha** is a React-based beginner app that helps you learn core React concepts.

## Technologies
- React (with functional components)
- JavaScript (ES6+)
- CSS (for styling)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Htooag/Yaycha.git
   cd Yaycha
   npm install
   npm run dev
`
